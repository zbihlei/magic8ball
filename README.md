## Installation

1. Clone the repository to your local machine:

    ```bash
    git clone https://github.com/zbihlei/magic8ball.git
    ```

2. Navigate to the project directory:

    ```bash
    cd yourproject
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Run dev mode:

    ```bash
    npm run dev
    ```
